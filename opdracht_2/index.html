Wat is het verschil met de script tag boven in te laden in de "head" tag t.o.v onder inladen aan het einde van je HTML tag

Als de script tag boven in de header wordt geladen, dan weet de script niet wat het moet veranderen,
terwijl als het beneden van de body wordt geladen, weet het wel wat het moet veranderen, omdat het al is geladen.

Wat doet de attribuut “defer” in de "script" tag

De defer attribuut zorgt ervoor dat de script tergelijketijd als het webpagina downloadt en activeert wanneer de pagina klaar is met laden.

Wat is het verschil tussen HTML en JavaScript?

HTML is het basisstructuur van een webpagina, en kan weinig functies gebruiken.
Javascript daartegenover is een programmeertaal die wordt gebruikt om functies en interactieve elementen toe te voegen aan een webpagina.
